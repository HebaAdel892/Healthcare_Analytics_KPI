# Saudi Healthcare Analytics Dashboard

## Project Overview
End-to-end healthcare analytics project analyzing hospital performance,
patient outcomes, and operational KPIs across 15 Saudi Arabian hospitals.
Built with Python, Power BI, and SQL, aligned with Saudi Vision 2030
healthcare transformation goals.

## Key Findings
- Analyzed 3,000+ admissions and 5,000+ treatments across 15 hospitals
- Calculated 12 industry-standard healthcare KPIs
- Identified Hajj season impact on hospital capacity (7.4% increase)
- Compared Government vs Private hospital performance
- Built interactive 4-page Power BI dashboard

## Tools & Technologies
- **Python**: pandas, matplotlib, seaborn, numpy
- **Power BI**: Interactive dashboard with DAX measures
- **SQL**: Complex queries with JOINs, CTEs, Window Functions
- **Excel**: Multi-sheet report with pivot tables

## Healthcare KPIs Tracked
| KPI | Value | Benchmark | Status |
|-----|-------|-----------|--------|
| Average Length of Stay | 7.97 days | 4-5 days | High |
| Bed Occupancy Rate | 78.7% | 80-85% | Low |
| 30-Day Readmission Rate | 14.9% | <10% | High |
| Patient Satisfaction | 3.86/5 | >4.0 | Good |
